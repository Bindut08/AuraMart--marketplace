AuraMarket — Premium Multi-Vendor Storefront
AuraMarket is a highly polished, luxury multi-vendor e-commerce platform and boutique fulfillment ecosystem. Built with a desktop-first, highly responsive design, it connects independent premium brands, bespoke studios, and artisan merchants directly with buyers.
The application features a secure customer shopping flow, live inventory management, and a dedicated Merchant Dispatch & Operations Portal where individual vendors can manage incoming orders, assign logistics carriers, and track fulfillment lifecycles.


🌟 Key Product Features
1. Curated Luxury Storefront
Dynamic Brand Showcase: High-fidelity product displays with pricing, brand attributions, and real-time inventory counts.
Fuzzy Search & Multi-Tag Filters: Instantly query products by keywords or narrow down catalogs using custom category tags (e.g., Audio, Ceramics, Apparel).
Aesthetic Shopping Cart Sidebar: Slide-out tray summarizing items, quantities, and calculated sub-totals.
2. Streamlined Secure Checkout
Two-Step Delivery Routing: Transition from shopping bag to delivery details where customers provide destination coordinates (Recipient Name and physical Address).
Automated Vendor Routing: Orders are automatically fragmented and dispatched directly into the isolated dashboards of the respective boutique merchants.
Live Stock Management: Completing checkout instantly decrements inventory levels across the catalog, preserving data consistency.
3. Dedicated Merchant Operations Portal
Inventory & Catalog Management: Create new listings with custom title, category tags, price in Indian Rupees (₹), image URLs, and initial stock quantities.
Orders & Shipments Fulfillment Office: A dedicated control center filtered strictly to show only items belonging to the currently logged-in vendor.
Multi-State Fulfillment Workflows: State-machine tracking for each item consignment:
Awaiting Review (Pending Acceptance)
Accepted & Packing (Preparing for dispatch)
In Transit (Logistics carrier assigned)
Delivered (Fulfillment complete)
Custom Transitor Assignment: Merchants can input specific shipping carriers (e.g., BlueDart, DHL, Aura Express) to track dispatches dynamically.
4. Live Customer Shipment Tracking Station
Transparent Timelines: A dedicated tracker at the bottom of the customer view displaying exactly which items are in transit, which courier is carrying them, and when they are delivered.

Styling Engine: Tailwind CSS

🚀 Getting Started Locally
Follow these instructions to run and test the application on your local machine.



Prerequisites
Ensure you have Node.js (version 18 or above) installed on your system.
1. Installation
Clone your repository, navigate to the root directory, and install the package dependencies:
code
Bash
# Install dependencies
npm install
2. Run the Development Server
Launch the Vite dev server to compile and run the application locally:
code
Bash
# Start development environment
npm run dev
The application will start, and the local URL will be printed to your terminal (typically http://localhost:3000). Open this in your browser to interact with the app.
